# 📖 Overview

MovieMatch is a modern AI-inspired movie discovery platform built using **React**, **Vite**, and the **TMDB API**. It helps users explore trending films, search for movies, discover cast members, analyze ratings, browse by genre, and save their favorite movies through a clean, responsive, and interactive interface.

Designed with a focus on performance and user experience, MovieMatch delivers fast navigation, engaging UI animations, and comprehensive movie information for cinema enthusiasts.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/saisharatkaruturi/MovieMatch.git
```

Navigate to the project

```bash
cd MovieMatch
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

# 🎬 Movie Recommender System

An AI-powered Movie Recommender web application that helps users discover movies based on popularity, genres, ratings, and personalized search. The application integrates real-time movie data using the TMDB API and provides an intuitive, responsive interface for exploring movies and viewing detailed information.

---

## 📌 Features

- 🔍 Search movies by title
- 🎞️ Browse trending and popular movies
- ⭐ View movie ratings and reviews
- 🎭 Explore movies by genre
- 📄 Detailed movie information page
- 🎬 High-quality movie posters and backdrops
- 📱 Responsive design for desktop and mobile
- ⚡ Fast and interactive user interface
- 🔗 Real-time movie data using TMDB API

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/movie-recommender.git
```

### Navigate into the project

```bash
cd movie-recommender
```

### Install dependencies

```bash
npm install
```

### Create Environment Variables

Create a `.env` file in the project root.

```env
VITE_TMDB_API_KEY=YOUR_TMDB_API_KEY
```

### Run the project

```bash
npm run dev
```
